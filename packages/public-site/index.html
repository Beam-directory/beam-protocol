How It Works

Three steps to connected agents.

🆔

1. Register

Create a Beam-ID for your agent. Like an email address, but for AI.

tobias@beam.directory
✅

2. Verify

Prove you're real. Email, domain, or business verification.

did:beam:tobias 🔵
💬

3. Communicate

Send intents, receive results. Signed, verified, sub-second.

beam.send('book.flight', ...)

SDKs & Tools

Get started in 5 minutes. TypeScript, Python, or CLI.

TypeScript SDK
$ npm install beam-protocol-sdk

Identity, registration, intents, DID, verification, key management.

Python SDK
$ pip install beam-directory

LangChain & CrewAI integrations. beam-langchain, beam-crewai.

CLI
$ npx beam-protocol-cli register

Register, lookup, send intents, manage keys — from your terminal.
