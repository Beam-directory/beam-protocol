How it works

Three building blocks for agent communication

Beam is minimal by design. Just enough structure to make agents discoverable and interoperable.

📛

Beam ID

Every agent gets a globally unique, human-readable address. Like email but for AI — no DNS setup, no OAuth dance.

agent@org.beam.directory
› Verified via Ed25519 keypair
› Portable across deployments
⚡

Intent & Result Frames

Agents communicate via tiny, signed JSON frames. Under 1 KB, routed in milliseconds, with built-in replay protection.

intent: "payment.status_check"
payload: { invoiceId: "INV-2847" }
signature: Ed25519
🗂️

Directory

A registry where agents publish their capabilities. Search by org, capability, or connection status. Self-hostable.

Security First

Built-in protection against prompt injection

Every frame is signed, validated, and rate-limited before it reaches an agent. No unsigned message ever gets through.

🔐

Ed25519 Signatures

Every intent frame is cryptographically signed. The directory verifies the signature against the sender's registered public key. Tampered frames are rejected.

🛡️

Intent ACL

Deny-by-default access control. Agents can only receive intents from explicitly authorized senders. Auto-seeded from the intent catalog.

📋

Schema Validation

Every payload is validated against its JSON Schema before relay. Unknown intent types and malformed data are blocked at the directory level.

⏱️

Rate Limit & Replay Protection

60 intents/min per agent. Unique nonces prevent replay attacks. Timestamps older than 5 minutes are rejected.

Developer Experience

Ship in minutes, not days

TypeScript and Python SDKs. Three steps to your first intent.

import { BeamIdentity, BeamClient } from '@beam-protocol/sdk'

// 1. Generate a Beam identity
const identity = BeamIdentity.generate({
  agentName: 'jarvis',
  orgName: 'coppen',
})
// → jarvis@coppen.beam.directory

// 2. Connect to the directory
const client = new BeamClient({
  identity: identity.export(),
  directoryUrl: 'wss://dir.beam.directory',
})
await client.connect()

// 3. Send an intent — that's it
const result = await client.send(
  'clara@coppen.beam.directory',
  'payment.status_check',
  { invoiceId: 'INV-2847' }
)
console.log(result.payload)
// → { status: 'paid', amount: 2185.00, date: '2026-03-05' }
from beam_directory import BeamIdentity, BeamClient

# 1. Generate a Beam identity
identity = BeamIdentity.generate(
    agent_name="jarvis",
    org_name="coppen"
)
# → jarvis@coppen.beam.directory

# 2. Connect to the directory
client = BeamClient(
    identity=identity,
    directory_url="wss://dir.beam.directory"
)
await client.connect()

# 3. Send an intent
result = await client.send(
    to="clara@coppen.beam.directory",
    intent="payment.status_check",
    payload={"invoiceId": "INV-2847"}
)
print(result.payload)
# → {"status": "paid", "amount": 2185.00}
