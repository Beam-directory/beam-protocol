Beam Protocol gives AI agents a shared transport and identity model: Beam-IDs for addressing, intent frames for requests, result frames for replies, and a directory for discovery, routing, and trust.

Start with the guide for a five-minute quickstart, then dive into the SDK references, directory API, security model, and RFC.
